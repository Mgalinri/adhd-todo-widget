# ADHD-Friendly To-Do List Widget 🎵

A beautiful, minimal desktop to-do list app designed for ADHD brains. Features Apple Music integration, one-task-at-a-time display, subtasks, time management, and a compact widget that stays on top.

## Features ✨

- **One Task at a Time**: Minimizes overwhelm by showing only the current task
- **Apple Music Integration**: See your now playing track while working
- **Time Management**: Set durations and timers for each task
- **Subtasks**: Break down complex tasks into manageable steps
- **Compact Widget Mode**: Minimal, desktop-friendly UI that pins to top
- **Always-on-Top**: Keep your widget above other windows
- **Duration Tracking**: Visual progress indicators

## Quick Start

```bash
bash setup.sh
source venv/bin/activate
python main.py
```

## Tech Stack

- **Frontend**: PyQt6 (Modern GUI)
- **Backend**: Python 3.9+
- **Apple Music**: AppleScript integration (macOS)
- **Database**: SQLite

## System Requirements

- macOS 10.14+
- Python 3.9+
- Apple Music app
